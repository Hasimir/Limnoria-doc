
.. _plugin-anonymous:

The Anonymous plugin
====================

.. command-do:

do <channel> <action>
^^^^^^^^^^^^^^^^^^^^^

Performs *<action>* in *<channel>*.


.. command-say:

say <channel|nick> <text>
^^^^^^^^^^^^^^^^^^^^^^^^^

Sends *<text>* to *<channel|nick>*. Can only send to *<nick>* if
supybot.plugins.Anonymous.allowPrivateTarget is True.
