
.. _plugin-internet:

The Internet plugin
===================

.. _command-whois:

whois <domain>
^^^^^^^^^^^^^^

Returns WHOIS information on the registration of *<domain>*.


.. _command-dns:

dns <host|ip>
^^^^^^^^^^^^^

Returns the ip of *<host>* or the reverse DNS hostname of *<ip>*.


.. _command-hexip:

hexip <ip>
^^^^^^^^^^

Returns the hexadecimal IP for that IP.
