
.. _plugin-quote:

The Quote plugin
================

.. _command-quote-random:

quote random [<channel>]
^^^^^^^^^^^^^^^^^^^^^^^^

Returns a random quote from *<channel>*. *<channel>* is only necessary if
the message isn't sent in the channel itself.
