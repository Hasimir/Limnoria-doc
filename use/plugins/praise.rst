
.. _plugin-praise:

The Praise plugin
=================

.. command-praise:

praise [<channel>] [<id>] <who|what> [for <reason>]
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Praises *<who|what>* (for *<reason>*, if given). If *<id>* is given, uses
that specific praise. *<channel>* is only necessary if the message isn't
sent in the channel itself.
