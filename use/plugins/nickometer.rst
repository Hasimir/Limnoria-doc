
.. _plugin-nickometer:

The Nickometer plugin
=====================

.. _command-nickometer-nickometer:

nickometer nickometer [<nick>]
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Tells you how lame said nick is. If *<nick>* is not given, uses the
nick of the person giving the command.
